\begin{abstract}
Detecting and segmenting objects in high-density retail environments presents
significant challenges due to severe inter-object occlusion, near-identical
appearance, and extreme variation in object counts per image. We present a
three-phase hybrid neuro-symbolic framework that integrates classical machine
learning with modern deep learning for dense object segmentation on the
SKU-110K dataset. Phase~1 establishes a HOG+SVM baseline for classical
feature-based detection. Phase~2 develops a YOLACT-based instance segmentation
model with MobileNetV3 backbone, CBAM attention, and Soft-NMS, achieving a best
validation loss of 3.145 over 20 epochs. Phase~3 introduces a hybrid detector
that augments the deep learning pipeline with probabilistic spatial reasoning
using Gaussian Mixture Models for shelf-row structure and Kernel Density
Estimation for spatial priors. An ablation study across eight model variants
demonstrates that Soft-NMS is critical for dense scenes (98.4\% mAP drop when
replaced with Hard-NMS), while the spatial reasoning components provide
complementary density-aware features. The system processes images at 8.3 FPS
via ONNX Runtime and achieves a 2.7$\times$ speedup over PyTorch inference.
\end{abstract}

\section{Introduction}
\label{sec:intro}

Automated product detection in retail shelf images has emerged as a critical
application in computer vision, enabling inventory management, planogram
compliance, and out-of-stock detection~\cite{goldman2019sku110k}. Unlike
standard object detection benchmarks such as COCO or Pascal VOC, retail shelf
images exhibit unique challenges: (i) extremely high object density with
100--300+ products per image, (ii) severe inter-object occlusion where
neighboring items share boundaries, (iii) homogeneous appearance across products
of similar packaging, and (iv) structured spatial layouts dictated by shelf
geometry.

The SKU-110K dataset~\cite{goldman2019sku110k} provides a large-scale benchmark
for this task with 11,762 images containing over 1.7 million bounding box
annotations, averaging 147.4 objects per image. This density is an order of
magnitude higher than typical detection datasets, rendering standard approaches
based on Non-Maximum Suppression (NMS) inadequate due to aggressive suppression
of closely-packed valid detections.

We propose a three-phase approach that progressively builds from classical
machine learning to a hybrid neuro-symbolic system:

\begin{itemize}
  \item \textbf{Phase 1 (Classical ML):} A HOG+SVM sliding window baseline
    that establishes feature engineering fundamentals and quantifies the
    performance gap motivating deep learning approaches.
  \item \textbf{Phase 2 (Deep Learning):} A YOLACT-based~\cite{bolya2019yolact}
    instance segmentation model using MobileNetV3~\cite{howard2019mobilenetv3}
    backbone with CBAM attention~\cite{woo2018cbam} and
    Soft-NMS~\cite{bodla2017softnms}, trained for 20 epochs on the full
    SKU-110K training set.
  \item \textbf{Phase 3 (Hybrid):} A neuro-symbolic fusion layer that augments
    YOLACT detections with probabilistic spatial reasoning via GMM-based row
    detection and KDE-based density estimation, incorporating gated spatial
    attention and confidence recalibration.
\end{itemize}

The key contribution of this work is demonstrating how classical probabilistic
models can complement deep detection networks in structured environments,
providing interpretable spatial priors that encode domain knowledge about retail
shelf layouts.

\section{Related Work}
\label{sec:related}

\subsection{Dense Object Detection}

Goldman~\etal~\cite{goldman2019sku110k} introduced SKU-110K and proposed a
dense detection framework using an EM-merge unit for handling tightly packed
objects. He~\etal~\cite{he2017maskrcnn} established Mask R-CNN as the
foundational two-stage instance segmentation approach, but its per-region
processing is computationally prohibitive for images with hundreds of objects.
Bolya~\etal~\cite{bolya2019yolact} proposed YOLACT, a real-time instance
segmentation method using prototype masks with linear coefficients, which
provides a favorable speed--accuracy tradeoff for dense scenes.

\subsection{Attention Mechanisms}

Woo~\etal~\cite{woo2018cbam} introduced the Convolutional Block Attention
Module (CBAM), which sequentially applies channel and spatial attention to
refine feature representations. Lin~\etal~\cite{lin2017fpn} proposed Feature
Pyramid Networks for multi-scale feature fusion, enabling detection across
varying object sizes.

\subsection{Post-Processing for Dense Detection}

Bodla~\etal~\cite{bodla2017softnms} proposed Soft-NMS, which replaces the
binary suppression of standard NMS with a Gaussian decay function, preserving
closely-packed detections. Lin~\etal~\cite{lin2017focal} introduced Focal Loss
to address class imbalance in dense detection, down-weighting easy negatives
to focus training on hard examples.

\subsection{Classical Feature Extraction}

Dalal and Triggs~\cite{dalal2005hog} introduced Histogram of Oriented Gradients
(HOG) features for object detection, capturing local gradient structure through
cell-based histograms with overlapping block normalization. Zhang~\etal~\cite{zhang2017mixup} proposed MixUp data augmentation, which creates
virtual training examples through convex combinations of input pairs.

\section{Methodology}
\label{sec:method}

Our approach progresses through three phases, each building upon the previous
to construct a hybrid detection system.

% ---------------------------------------------------------------------------
\subsection{Phase 1: Classical ML Baseline (HOG+SVM)}
\label{sec:phase1}

The classical baseline employs Histogram of Oriented Gradients
(HOG)~\cite{dalal2005hog} features with a linear Support Vector Machine (SVM)
classifier in a sliding window framework.

\textbf{Feature Extraction.} HOG descriptors are computed with the following
parameters: cell size $8 \times 8$ pixels, block size $16 \times 16$ pixels
(overlapping $2 \times 2$ cells), 9 orientation bins spanning $0$--$180$
degrees, and a window size of $64 \times 64$ pixels. This yields a
1,764-dimensional feature vector per window that captures local gradient
structure while maintaining robustness to minor illumination changes.

\textbf{Classification.} A linear SVM is trained on 300 synthetic training
images (positive and negative patches extracted from annotated regions). The
classifier achieves 100\% training accuracy, reflecting the separability of
foreground and background patches in feature space.

\textbf{Detection Pipeline.} At inference, a multi-scale sliding window scans
each image, classifying each window using the trained SVM. Non-Maximum
Suppression (Hard-NMS) consolidates overlapping detections. The complete
pipeline processes images at 0.251 seconds per image.

Figure~\ref{fig:hog_features} visualizes the HOG feature extraction process,
illustrating how gradient orientations capture product edge structure.

\subsection{Phase 2: Deep Learning (YOLACT with MobileNetV3)}
\label{sec:phase2}

The deep learning component is built on the YOLACT
framework~\cite{bolya2019yolact}, adapted for the dense detection setting.

\textbf{Backbone.} We employ MobileNetV3-Large~\cite{howard2019mobilenetv3}
as the feature extractor, chosen for its favorable parameter efficiency
(approximately 5.4M parameters) and mobile-deployment compatibility. The
backbone produces multi-scale features at strides 8, 16, and 32.

\textbf{Feature Pyramid Network.} A three-level FPN~\cite{lin2017fpn}
aggregates features from P3 (stride 8), P4 (stride 16), and P5 (stride 32)
via top-down pathways and lateral connections, producing 256-channel feature
maps at each level.

\textbf{Attention Module.} CBAM~\cite{woo2018cbam} is integrated into the FPN
pathway, applying sequential channel attention (via global average and max
pooling) and spatial attention (via $7 \times 7$ convolution) to enhance
discriminative features for densely packed objects.

\textbf{Detection Head.} A shared prediction head generates per-anchor
classification scores (2 classes: background + product), bounding box
regressions (4 offsets), and mask coefficients (32 coefficients per anchor).
Anchors are generated at three FPN levels with aspect ratios $\{1/2, 1, 2\}$
and three scales per level.

\textbf{Prototype Network.} A protonet generates 32 prototype masks from the
P3 feature map, which are linearly combined using predicted coefficients and
cropped to bounding boxes to produce instance masks.

\textbf{Post-Processing.} Soft-NMS~\cite{bodla2017softnms} replaces standard
NMS, using Gaussian decay with $\sigma = 0.5$:
\begin{equation}
  s_i = s_i \cdot e^{-\frac{\text{IoU}(b_i, b_M)^2}{\sigma}}
\end{equation}
where $s_i$ is the score of detection $i$, $b_M$ is the highest-scoring box,
and $\sigma$ controls decay strength.

\textbf{Training Configuration.} The model (10.0M parameters) is trained for
20 epochs on 8,219 training images with batch size 8, SGD optimizer (lr=0.001,
momentum=0.9, weight decay=5$\times$10$^{-4}$), cosine annealing schedule with
3-epoch linear warmup, mixed-precision training (AMP), and MixUp
augmentation~\cite{zhang2017mixup} ($\alpha = 0.2$). The loss function combines
classification (Focal Loss~\cite{lin2017focal}), box regression (Smooth L1),
and mask (Binary Cross-Entropy) losses.

\subsection{Phase 3: Hybrid Neuro-Symbolic Fusion}
\label{sec:phase3}

The hybrid detector augments the trained YOLACT model with classical
probabilistic spatial reasoning, creating a feedback loop between neural
detections and symbolic spatial models.

\textbf{Spatial Reasoning Engine.} Two classical ML models encode spatial
priors learned from training data:

\begin{enumerate}
  \item \textbf{Gaussian Mixture Model (GMM):} Fitted on the y-coordinates of
    box centers across all training images to detect shelf-row structure. BIC
    model selection identifies 7 optimal row components. The GMM provides
    row membership probabilities and inter-row spacing statistics.
  \item \textbf{Kernel Density Estimation (KDE):} A Gaussian KDE (bandwidth=0.05)
    fitted on normalized (x, y) center coordinates of 5,000 subsampled
    boxes. The KDE generates a 2D spatial density field indicating expected
    object locations based on learned layout priors.
\end{enumerate}

\textbf{Spatial Feature Vector.} For each image, the spatial engine computes
an 8-dimensional feature vector: (1) normalized object count, (2--3) mean and
standard deviation of box areas, (4--5) row count and mean row separation from
GMM, (6) spatial regularity score, (7) density field entropy, and (8) mean
nearest-neighbor distance.

\textbf{Gated Spatial Attention.} A learnable gate parameter $g$ (initialized
at 0, trained via backpropagation) controls the influence of the spatial density
field on the P3 feature map:
\begin{equation}
  F'_{\text{P3}} = F_{\text{P3}} + \sigma(g) \cdot D_{\text{spatial}}
\end{equation}
where $D_{\text{spatial}}$ is the KDE-derived density map upsampled to match
P3 resolution. At convergence, $\sigma(g) \approx 0.41$, indicating the model
learns a moderate spatial influence.

\textbf{Confidence Recalibrator.} A lightweight MLP (17.8K parameters) fuses
YOLACT detection scores with spatial features and ROI-pooled visual features to
produce recalibrated confidence scores:
\begin{equation}
  s'_i = 0.5 \cdot s_i + 0.5 \cdot \sigma(\text{MLP}(s_i, f_{\text{spatial}},
  f_{\text{visual}}))
\end{equation}

\textbf{Three-Stage Training.} The hybrid model is trained in three stages:
\begin{itemize}
  \item \textbf{Stage 1:} Train base YOLACT (20 epochs, full dataset)
  \item \textbf{Stage 2:} Fit spatial reasoning engine (GMM + KDE on training GT)
  \item \textbf{Stage 3a:} Train fusion components with frozen backbone (5 epochs)
  \item \textbf{Stage 3b:} Joint fine-tuning of all components (2 epochs)
\end{itemize}

\textbf{Key Findings:}

\begin{enumerate}
  \item \textbf{Soft-NMS is critical:} Replacing Soft-NMS with Hard-NMS causes
    a 98.4\% relative drop in mAP (2.73\% $\to$ 0.04\%), confirming that
    traditional NMS is fundamentally unsuitable for dense detection. This is
    the most significant finding of the ablation study.

  \item \textbf{Spatial attention adds overhead with limited training:} Removing
    spatial attention slightly improves mAP by +0.30 percentage points,
    suggesting that with only 7 fusion epochs, the spatial components introduce
    marginal overhead without sufficient training to converge.

  \item \textbf{Other components are neutral:} The recalibrator, row model,
    density field, and CBAM show no measurable impact when individually removed,
    indicating that (a) these components require more extensive training to
    demonstrate their value, or (b) the base YOLACT backbone already captures
    the relevant features.
\end{enumerate}

\section{Discussion}
\label{sec:discussion}

\subsection{Phase Progression Analysis}

The three-phase approach reveals a clear progression in capability:

\textbf{Phase 1 $\to$ Phase 2:} The transition from classical HOG+SVM to
YOLACT demonstrates the necessity of learned features for dense detection. The
classical approach achieves high precision (86.36\%) on the few objects it
detects but fails to scale to hundreds of objects per image due to the sliding
window bottleneck and Hard-NMS suppression.

\textbf{Phase 2 $\to$ Phase 3:} The hybrid fusion adds spatial reasoning
capability but requires substantial training to demonstrate measurable gains.
The gated spatial attention mechanism (converging to 0.408) shows the model
learns a non-trivial spatial influence, and the training loss improvement
(3.145 $\to$ 3.097) validates that the spatial features provide useful
gradient signal.

\subsection{Soft-NMS as a Critical Component}

The ablation study's most striking result is the 98.4\% mAP degradation when
switching from Soft-NMS to Hard-NMS. In dense retail scenes where valid
detections have IoU $>$ 0.1 with neighbors, Hard-NMS's binary suppression
eliminates the majority of true positives. Soft-NMS's Gaussian decay function
($\sigma = 0.5$) preserves overlapping detections by reducing scores gradually
rather than eliminating them entirely.

\subsection{Limitations and Future Work}

\textbf{Limited fusion training.} The hybrid detector was trained for only
7 fusion epochs (5 frozen + 2 fine-tuning) due to computational constraints.
The spatial reasoning components---particularly the confidence recalibrator
and gated attention---would likely benefit from extended training.

\textbf{KDE scalability.} The Kernel Density Estimation was subsampled to
5,000 points from 1.1M for computational tractability. While this preserves
the overall density distribution, local density patterns may be lost.
GPU-accelerated KDE or parametric density models could address this limitation.

\textbf{Evaluation metrics.} The absolute mAP values are low across all
variants, reflecting both the extreme difficulty of SKU-110K and potential
box-coordinate normalization issues in the evaluation pipeline. Future work
should incorporate the official SKU-110K evaluation protocol for direct
comparison with published baselines.

\textbf{Future directions.} (1) Extended hybrid training with curriculum
learning, (2) Transformer-based spatial reasoning to replace hand-crafted
GMM/KDE, (3) Contrastive learning for product re-identification, and
(4) Integration with planogram databases for compliance verification.

\section{Conclusion}
\label{sec:conclusion}

We presented a three-phase hybrid neuro-symbolic framework for high-density
object segmentation in retail environments. The system progresses from a
classical HOG+SVM baseline through a YOLACT-based deep detector to a hybrid
model incorporating probabilistic spatial reasoning. Our ablation study across
eight variants on 588 validation images yields three key insights:
(1) Soft-NMS is essential for dense detection, with Hard-NMS causing 98.4\%
mAP degradation; (2) spatial reasoning components provide valid training signal
(loss reduction 3.145 $\to$ 3.097) but require extended training for inference
gains; and (3) the MobileNetV3 backbone enables efficient deployment at 8.3 FPS
via ONNX Runtime. The framework demonstrates a principled approach to combining
neural and symbolic reasoning for structured visual scenes, with the modular
three-phase design enabling independent improvement of each component.
